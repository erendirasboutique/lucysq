# Lucy's Quinceañera Branded Website

A luxury Next.js / Vercel-ready website mockup using Lucy's logo and LT Recoleta font.

## Deploy
1. Upload this folder to GitHub.
2. Import it into Vercel.
3. Framework Preset: Next.js.
4. Deploy.

## Assets
- Logo: `public/logo.png`
- Font: `public/fonts/lttrecoleta.otf`
- Add photos/video later in `public/images` or `public`.

## Customize
Edit `app/data.js` for venue, schedule, RSVP URL, hotels, and FAQ.
