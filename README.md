# Lucy's Quinceañera Branded Site

Production-ready Next.js mockup for Lucy's Quinceañera.

## Updated Travel Content

This version includes Rincón de Guayabitos, Nayarit, Mexico content for:

- Recommended hotels
- Travel from Puerto Vallarta International Airport (PVR) to Guayabitos
- Transportation options
- Weekend guide
- Venue/parking placeholders
- Things to do nearby

## Setup

1. Upload this project to GitHub.
2. Connect the repository to Vercel.
3. Add your own photos/videos inside `public/images` when ready.
4. Add the LT Recoleta font file to `public/fonts/lttrecoleta.otf` if it is not already present.

## Commands

```bash
npm install
npm run dev
```

## Notes

Hotel availability, prices, transportation costs, and travel times may change. Update final booking links and exact venue address once confirmed.
